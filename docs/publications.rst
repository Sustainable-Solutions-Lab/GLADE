Publications
============

This page lists peer-reviewed papers and preprints based on GLADE. If you have
published work that uses the model, please open a pull request or get in touch
so we can add it here.

Citing GLADE
------------

If you use GLADE in your research, please cite the paper that introduces and
applies the model:

   van Greevenbroek, K., Davis, S. J., & Caldeira, K. (2026).
   *Reducing health and climate impacts of the global food system*.
   arXiv:2606.15471. https://doi.org/10.48550/arXiv.2606.15471

.. code-block:: bibtex

   @article{vangreevenbroek2026glade,
     title   = {Reducing health and climate impacts of the global food system},
     author  = {van Greevenbroek, Koen and Davis, Steven J. and Caldeira, Ken},
     year    = {2026},
     journal = {arXiv preprint arXiv:2606.15471},
     doi     = {10.48550/arXiv.2606.15471},
     url     = {https://arxiv.org/abs/2606.15471}
   }

To cite a specific version of the software itself, use the metadata in
``CITATION.cff`` at the repository root, also exposed via the
`Cite this repository <https://github.com/Sustainable-Solutions-Lab/GLADE>`_
button on GitHub.
